# Hybrid Fitness Lifestyle Growth Lab

Senior social media intelligence analyst command center codifying the playbook for scaling a hybrid fitness/lifestyle/entrepreneur brand to 100K+ followers in 10 months.

## Stack
- Next.js 14 (App Router, TypeScript)
- Tailwind CSS with custom theme tokens
- Static generation for fast deploys on Vercel

## Getting Started
```bash
npm install
npm run dev
```
Visit `http://localhost:3000` to explore the growth lab interface.

## Available Scripts
- `npm run dev` – start the development server
- `npm run build` – create a production build
- `npm run start` – run the production server locally
- `npm run lint` – lint the codebase

## Structure Highlights
- `src/app/page.tsx` – renders the intelligence dashboard, roadmap, and tactical breakdowns
- `src/data` – structured intelligence for target accounts, roadmap phases, and operating systems
- `src/app/globals.css` & `tailwind.config.ts` – theme and design primitives

## Deployment
Ensure the project builds locally (`npm run build`). Deploy with Vercel CLI as configured in the project brief.
